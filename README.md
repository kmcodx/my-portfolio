# Khurram Malik Portfolio (Next.js 14)

Premium dark portfolio landing page built with:

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- ShadCN-inspired UI primitives

## Sections

- Hero
- About timeline
- Skills cards
- Experience accordion + counters
- Project showcase
- Architecture philosophy
- Contact form

## Run

```bash
npm install
npm run dev
```
